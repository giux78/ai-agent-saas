## Example

```bash
curl -X 'POST' \
  'https://qod.io/openapi/zefiro/v0.5/generate' \
  -H 'accept: application/json' \
  -H 'X-Auth: YOURAPIKEY' \
  -H 'Content-Type: application/json' \
  -d '[
  {
    "content": "Sei un assistente onesto e rispettoso",
    "role": "assistant",
    "id" : "sys"

  },
  {
    "content": "Mi consigli una ricetta per la cena",
    "role": "user"
  }
]'
```
